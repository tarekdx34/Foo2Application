# 🃏 فووق

لعبة كلمات جماعية بالبطاقات — تحدّي الذاكرة والسرعة والإبداع!

---

## 🎯 فكرة اللعبة

**فووق** لعبة بسيطة وممتعة تُلعب بمجموعة من الأصدقاء أو العيلة.  
كل دور، يسحب اللاعب بطاقة تحتوي على:

- **السؤال** — فئة أو وصف معيّن (مثل: *لاعب كورة* أو *دولة عربية* أو *فيلم مصري*)
- **الشرط** — قيد يجب أن ينطبق على الإجابة (مثل: *يبدأ بحرف "س"* أو *من 5 حروف أو أكتر*)

على اللاعب أن يُجيب بكلمة تنطبق على **السؤال** و**الشرط** معًا قبل انتهاء الوقت.

---

## 🕹️ طريقة اللعب

1. **إعداد اللعبة** — اختر الفئات التي تريد اللعب بيها، عدد البطاقات، والوقت المسموح لكل سؤال.
2. **سحب البطاقة** — يسحب اللاعب الحالي بطاقة من الكومة.
3. **قراءة السؤال** — يظهر وجه البطاقة بالسؤال (مثلاً: *ممثل أمريكي*).
4. **الشرط** — عند قلب البطاقة يظهر الشرط العشوائي (مثلاً: *يحتوي على حرف "ر"*).
5. **الإجابة** — يجب على اللاعب إيجاد إجابة صحيحة قبل انتهاء الوقت.
6. **التقييم** — يقرر باقي اللاعبين إذا كانت الإجابة صحيحة أم لا.
7. **الدور التالي** — تنتقل البطاقة للاعب التالي.

## 🔤 الشروط

تنقسم الشروط إلى نوعين:

- **شرط حرفي** — مرتبط بحرف عشوائي من الأبجدية العربية (مثل: *يبدأ بحرف "ط"*)
- **شرط ثابت** — قيد محدد (مثل: *اسمه مكوّن من كلمتين*)

أمثلة على الشروط:
- يبدأ بحرف "..."
- ينتهي بحرف "..."
- يحتوي على حرف "..."
- مفيهوش حرف "..."
- فيه حرف مكرر
- من 3 حروف بس
- من 5 حروف أو أكتر
- اسمه مكوّن من كلمتين

## 📱 تشغيل اللعبة

### متطلبات التشغيل
- Node.js v18 أو أحدث
- npm

### خطوات التثبيت

```bash
# تثبيت الاعتماديات
npm install

# تشغيل وضع التطوير
npm run dev

# بناء النسخة النهائية
npm run build

# معاينة النسخة النهائية
npm run preview
```

### تشغيل بدون إنترنت (PWA)

اللعبة تدعم التثبيت على الجهاز والتشغيل بدون إنترنت:
1. افتح اللعبة في المتصفح (Chrome أو Edge)
2. اضغط على زر **"تثبيت اللعبة"** في الصفحة الرئيسية
3. بعد التثبيت يمكن تشغيلها مباشرة من سطح المكتب أو الهاتف
